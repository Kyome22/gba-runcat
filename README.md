# RunCat endless game for GBA

RunCat endless game for GBA (Game Body Advance). This project is written by Embedded Swift.

<img src="https://github.com/user-attachments/assets/81b04c75-4564-4e5b-8844-1fa72d9a2d03" width="480" />

## Requirements

- Language: Swift 6.2.3
- Toolchain: [gba-llvm-devkit](https://github.com/stuij/gba-llvm-devkit) release-v1
- Target: ARMv4T
- Emulator: mGBA 0.10.5

## How to build

```sh
make
```

## How to run on the emulator

```sh
make run
```

## Play on actual GBA hardware

You can play on a real GBA by flashing the RunCat.gba file to a cartridge ([product of InsideGadgets](https://shop.insidegadgets.com/product/gba-4mb-rom-only-flash-cart/)) using [Lesserkuma/FlashGBX](https://github.com/Lesserkuma/FlashGBX).
